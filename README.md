The repo represents the data, RMarkdown build files, and final manuscript "Evaluation of Ground-based Smoke Sensors for Wildfire Detection and Monitoring in Canada" which has been submitted to the journal Fire.

The repo is also archived on Zenodo at: https://doi.org/10.5281/zenodo.18202275

See https://github.com/nrcan-cfs-fire/Ground-smoke-sensors/blob/main/Evaluation-of-ground-based-smoke-sensors-for-wildfire-detection-in-Canada.pdf for the manuscript pdf.
